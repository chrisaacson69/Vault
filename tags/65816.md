# 65816
> The WDC 65C816 — the SNES CPU: 16-bit A/X/Y toggled by the M/X flags, 24-bit addressing, bank-relative code. The 6502's big sibling; a disassembler must track M/X width to decode immediates.

- [Breaking Down an SNES Cart — the teardown method](../research/gaming/snes-cartridge-teardown.md) — why the NES 6502 disassembler does not transfer as-is; RESET vector as the entry point
- [SNES teardown — `snes-decompiler`](../projects/game-annotation/snes/README.md) — the substrate that will host an M/X-width-aware 65C816 disassembler when a target lands
- [Nobunaga's Ambition (SNES) compiled to native 65816](../research/gaming/na1-snes-native-port.md) — a KOEI title compiled straight to native 65C816 (compiler-generated cdecl: `PEA`/`JSL`/`PLA`); drove a native 65816→C decompiler + a 32-bit mul/div lib finding
