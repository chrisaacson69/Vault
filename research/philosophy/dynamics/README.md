**Trunk:** [Trunk 4 — Structural Dynamics](../the-four-trunks.md#trunk-4--structural-dynamics). Working name not yet settled. Candidate territory: *opposing-forces structure*, *irreducibility dynamics*, *gradient structure*, *structural tension*.

## Core thesis

Every concentration of power generates counter-forces. Every governing system faces Gödel-shaped walls. Every attempt to organize computationally irreducible agents generates reaction. Every civilizational arc follows recognizable rise-and-fall patterns. These aren't coincidences — they're the same structural principle at different scales.
