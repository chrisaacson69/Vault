# Batch Resize
> CLI tool for batch resizing images with configurable size/scaling modes.

**Repo:** `https://github.com/chrisaacson69/batch-resize`

## Overview

Python utility for batch resizing images using PIL. Supports fixed size, percentage scaling, and aspect-ratio-preserving modes. Reads from an input directory, writes to an output directory with configurable quality and naming.
