# YouTube Migration
> Turning vault research into YouTube scripts — putting ideas into the ether and letting them fall where they do.

**Repo:** [github.com/chrisaacson69/youtube-migration](https://github.com/chrisaacson69/youtube-migration)
**Tools:** [animation-studio](https://github.com/chrisaacson69/animation-studio)
**Infrastructure:** [animation-studio-pod](https://github.com/chrisaacson69/animation-studio-pod)

## Vault ↔ Project Relationship

The Vault's research pages contain the **ideas and theory**. The youtube-migration repo contains the **episode scripts and production assets**. The Vault informs the scripts; production discoveries feed back into the Vault.

**The rule:** Ideas and theory live in the Vault. Episode scripts and production assets live in the repo. Cross-link, don't duplicate.

Content pipeline: Vault research → episode script → voiceover (sets timing) → Manim scenes → character art → composite → publish.

## Channel Strategy

Single channel with playlists. The connections between topics are the differentiator. Anyone can talk about insurance or morality separately — the synthesis is what's new.
